기술 역량

프론트엔드

HTML5 & CSS3

저는 HTML과 CSS를 위에서 언급한 부산직업전문학교의 반응형 웹 디자인 수업과 W3Schools에서 배웠습니다. 기본 웹 사이트를 문제 없이 만들 수 있고, 각 기기에 반응해 자동으로 크기를 맞추는 반응형 웹 사이트를 어떻게 만들어야 할지 압니다.

JavaScript

저는 자바스크립트를 학원 과정을 마친 뒤 바로 시작해 FreeCodeCamp에서 3개월 간 배웠습니다. 'JavaScript Algorithms and Data Structures Certification' 섹션에서 'Intermediate Algorithm Scripting'과 'JS Algorithms and Data Structures Projects'를 제외하고 모두 완료했습니다. 또한 저는 'Data Visualization with D3'와 'JSON APIs and Ajax'도 완료했습니다. 자바스크립트의 기본을 잘 알고 자바스크립트 코드를 웹 사이트에 적용할 수 있습니다.

Bootstrap & jQuery

저는 부트스트랩과 제이쿼리를 FreeCodeCamp에서 약 한 달간 배웠습니다. 'Front End Libraries Certification' 섹션에 있던 부트스트랩과 제이쿼리 파트를 완료했습니다. 부트스트랩 템플릿을 어떻게 써야 하는지 알고 제이쿼리 코드를 웹 사이트에 적용할 수 있습니다.


백엔드

PHP & MySQL

저는 PHP와 MySQL을 TutorialRepublic에서 3개월 조금 넘게 배웠습니다. PHP의 기본 개념을 알고 데이터베이스를 어떻게 만들어서 연결하는지, 테이블은 어떻게 만드는지 압니다. CRUD 시스템, 로그인 시스템을 이해할 수 있습니다. 최근에 PHPMailer 패키지를 이용해 Gmail SMTP로 연락 폼에서 이메일을 보내는 방법을 배워서 바로 제 포트폴리오 웹사이트에 적용시켜 보았습니다.

Python & Django

파이썬은 여러 외국 사이트에서 약 2개월동안 배웠습니다. 가위바위보 게임, 점 쳐 주는 게임, 행맨 등 배우는 과정에서 간단한 게임을 만든 적이 있습니다. 장고에 대해서 말씀드리면 저는 장고 공식 웹 페이지에서 장고 웹 애플리케이션 튜토리얼을 따라 만든 적이 있습니다. 그 과정에서 리눅스 환경에서 가상 환경을 만들어 패키지를 관리하는 법을 배웠습니다.
