My Portfolio Website

I started making this from June 12th, 2020.

View on Github

My Goal

To make a portfolio that looks simple and modern.

Technologies

  • Front-End: HTML, CSS, and JavaScript
  • Back-End: PHP

Features

  • In the index, users should be able to look at a summary of all the pages.
  • In Resume, users should be able to look at my resume with a short introduction of myself, education status, and my skills.
  • In Portfolio, users should be able to view my projects I put in, one by one by clicking "view project".
  • In Contact, users should be able to send an email to me via the contact form.

Progress

  • Used Bootstrap template for overall website design. Black theme.
  • Removed unnecessary social media buttons, address, phone number, and Google Maps.
  • Added my three main qualities right below of the introduction. Added self-taught part on Education. Removed unnecessary animated graphs, replaced them with more detailed explanations of what I can do.
  • Used PHPMailer codes to enable sending emails on contact form because it's more secure than simple PHP codes. Downloaded PHPMailer through Composer.
  • SMTP servers are mostly paid services, so I used Google's SMTP since it's free. Went to Google Developer Console page, made a project called "PHPMailer", enabled Gmail API. I made a new client ID, put "http://localhost/phpmailer/get_oauth_token.php". I put my credentials in get_oauth_token.php file, ran it, got my refresh token, and applied to PHPMailer OAuth2 code.
  • Used XAMPP localhost server to test the contact form. It worked but can't customize the sender address. Added "AddReplyTo()" function to substitute "setFrom()" function to display the sender's address.
  • Copied .html files to the folder "kr" to make Korean translated pages. While making Korean pages, I made a languages selector on the left-top of the index.
