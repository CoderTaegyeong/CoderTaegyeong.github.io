My Portfolio Website

I started making this from June 12th, 2020.

View on Github

My Goal

To make a portfolio that looks simple and modern.

Technologies

  • Front-End: HTML, CSS, and JavaScript
  • Back-End: PHP

Features

  • In the index, users should be able to look at a summary of all the pages.
  • In Resume, users should be able to look at my resume with a short introduction of myself, education status, and my skills.
  • In Portfolio, users should be able to view my projects I put in, one by one by clicking "view project".
  • In Contact, users should be able to send an email to me via the contact form.

Progress

  • Use Bootstrap template for overall website design. Black theme.
  • Remove unnecessary social media buttons, address, phone number, and Google Maps.
  • Add my three main qualities right below of the introduction. Add self-taught part on Education. Remove unnecessary animated graphs, replace them with more detailed explanations of what I can do.
  • Use PHPMailer codes to enable sending emails on contact form because it's more secure than simple PHP codes.
  • Go to Google API page and get authenticated by OAuth2 to use PHPMailer OAuth2 code.
  • Use XAMPP localhost server to test the contact form. It works but can't customize the sender address. Add "AddReplyTo()" function to substitute "setFrom()" function to display the sender's address.
